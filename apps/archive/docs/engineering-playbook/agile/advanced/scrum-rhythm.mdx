# Scrum Rhythm

<table>
  <thead>
    <tr>
      <th>Activity</th>

      <th>Accountable</th>

    </tr>
  </thead>

    <tr>
      <td>Project Standup</td>
      <td>Tue-Fri 9AM</td>
      <td>15 min</td>
      <td>Everyone</td>
      <td>Process Lead</td>
      <td>What has been accomplished, next steps, blockers</td>
    </tr>
    <tr>
      <td>Sprint Demo</td>
      <td>Monday 9AM</td>
      <td>1 hour</td>
      <td>Everyone</td>
      <td>Dev Lead</td>
      <td>Present work done and sign off on user story completion</td>
    </tr>
    <tr>
      <td>Sprint Retro</td>
      <td>Monday 10AM</td>
      <td>1 hour</td>
      <td>Everyone</td>
      <td>Process Lead</td>
      <td>Dev Teams share learnings and what can be improved</td>
    </tr>
    <tr>
      <td>Sprint Planning</td>
      <td>Monday 11AM</td>
      <td>1 hour</td>
      <td>Everyone</td>
      <td>PO</td>
      <td>Size and plan user stories for the sprint</td>
    </tr>
    <tr>
      <td>Task Creation</td>
      <td>After Sprint Planning</td>
      <td>-</td>
      <td>Dev Team</td>
      <td>Dev Lead</td>
      <td>Create tasks to clarify and determine velocity</td>
    </tr>
    <tr>
      <td>Backlog Refinement</td>
      <td>Wednesday 2PM</td>
      <td>1 hour</td>
      <td>Dev Lead, PO</td>
      <td>PO</td>
      <td>Prepare for the next sprint and ensure that stories are ready for the next sprint</td>
    </tr>
